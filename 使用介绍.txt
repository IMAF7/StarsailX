一、软件是什么
--------------------------------------------------------------------------------
StarsailX 是一款在 Windows 上运行的 Starsail 多账号管理工具。

主要能力：
  · 在一个窗口里管理多个 Starsail 账号（独立登录态，互不串号）
  · 左侧账号列表 + 右侧 Starsail 网页（Edge WebView2 内核）
  · 未读角标、Starsail 提示音、置顶、备注、重新登录等

数据默认保存在：D:\StarsailX（可用环境变量 STARSAILX_DATA_ROOT 修改）


二、系统与运行环境
--------------------------------------------------------------------------------
【必需】
  · Windows 10 / 11，64 位系统
  · 官方安装包 StarsailX_Setup_*.exe 或便携版 StarsailX_纯净版
  · Microsoft Edge WebView2 Runtime 64 位
  · 建议 D 盘可用（默认数据目录 D:\StarsailX）

【启动方式】
  · 安装版：开始菜单 / 桌面快捷方式 → StarsailX.exe
  · 便携版：解压后运行 StarsailX.exe

【可选环境变量】
  STARSAILX_DATA_ROOT              数据根目录，默认 D:\StarsailX
  STARSAILX_ENGINE                 页面引擎，默认 webview2
  STARSAILX_WEBVIEW2_BROWSER_FOLDER   固定版 WebView2 运行时目录

（仍兼容旧名 TEAMSX_DATA_ROOT / TEAMSX_ENGINE，建议改用 STARSAILX_*）


三、界面布局
--------------------------------------------------------------------------------
┌─────────────────────────────────────────────────────────────┐
│  StarsailX 标题栏                                            │
├──────────────┬──────────────────────────────────────────────┤
│  账号列表    │  Starsail 聊天主界面                          │
│  · 搜索      │                                               │
│  · 添加/刷新 │                                               │
└──────────────┴──────────────────────────────────────────────┘

【圆点状态】
  绿色 — 已登录且活跃
  黄色 — 已登录但休眠
  红色 — 未登录 / 失败

【红色数字角标】未读消息数


四、账号管理
--------------------------------------------------------------------------------
4.1 首次使用
  · 无账号时显示欢迎登录页，输入账号密码登入
  · 或使用侧栏「添加」手动添加账号

4.2 右键已登录账号
  · 编辑备注 / 编辑账号 / 置顶账号 / 重新登录 / 删除账号

4.3 通知
  · 托盘右键可开关消息提示音
  · 提示音文件：audio\Starsail.mp3


五、数据目录（D:\StarsailX）
--------------------------------------------------------------------------------
  accounts.txt          账号备份文本
  db\                   数据库
  profiles\             各账号浏览器配置
  cache\                缓存

删除 D:\StarsailX 将清空全部本地数据，请谨慎操作。
